Frontend (Expo) - instruções iniciais

Este diretório contém o scaffold inicial do app React Native (apenas componentes básicos).

Para iniciar um projeto Expo real, execute no diretório raiz:

npm install -g expo-cli
expo init frontend

Depois copie os arquivos deste diretório para o projeto criado e instale dependências como `react-navigation` e `firebase`.

- App.js
- src/screens/StudentHome.js
- src/components/ProductCard.js
